## Codejock Suite Pro 到底是个啥

简单说，它是 Codejock Software 公司给 ActiveX / COM 开发者打包的一整套 Windows 桌面 UI 组件库。公司 1998 年成立，专门干这事，已经有二十多年了。这个 Suite Pro 是把 Codejock 几乎所有 ActiveX 控件捆一起，一次给你 **11 个核心组件**：

- **Calendar** — Outlook 式日历与排程控件，支持日 / 工作周 / 周 / 月视图，绑定 SQL Server、MySQL、XML、Outlook 数据

- **Chart Pro** — 图表与数据可视化，2D / 3D、饼图柱图折线图、金融图、雷达图、漏斗、甘特一应俱全

- **Command Bars** — Office 风格 Ribbon、菜单、工具栏、弹出画廊、Backstage 视图

- **Controls** — 按钮、组合框、对话框、编辑框、Tab 控件、列表视图、树视图、进度条等基础控件

- **Docking Pane** — Visual Studio 风格的可停靠 / 浮动 / 标签 / 自动隐藏窗格

- **Property Grid** — VS 式属性编辑网格，支持布尔、颜色、日期、字体等内置类型

- **Report Control** — Outlook 式报表控件，分组、合并单元格、虚拟模式、字段选择器

- **Shortcut Bar** — Outlook 式导航窗格

- **Skin Framework** — 应用级皮肤框架，内置 Windows 10 / 8、Office 2010 等多种主题

- **Syntax Edit** — IDE 式语法编辑器，代码高亮、行号、折叠、断点、IntelliSense

- **Task Panel** — Office / Explorer 风格任务面板

说白了，**Office 风格 + Visual Studio 风格的现代 Windows 桌面 UI，这一套基本包圆了**。老张那堆零碎控件，全在这 11 个里。

## v24.3.0：2025 年底刚更新的版本

2025 年 12 月 23 日，Codejock 同步发布了 Toolkit Pro（MFC 版）和 Suite Pro（ActiveX 版）的 **v24.3.0**，这次更新对国内开发者比较有用的几个点：

- **RTL（从右到左）布局支持**，中东地区本地化项目能直接用了

- **30+ 种语言**的多语言框架进一步完善

- **Skin Framework** 对 Windows 8 / 10 系统皮肤做了改进

- **WebBrowser 控件接入 WebView2**，可以指定 `UserDefaultFolder`，等于把老 ActiveX 程序里的内嵌浏览器换成了现代内核

- **Report Control** 新增 `SetGridControl` 等接口，报表嵌套更灵活

如果你买的是一年更新版，这几个改进是免费拿到的；30 天更新版的用户可以等下一个大版本时再考虑续费。

## 谁该认真考虑这套

我列一下我对号的判断：

- **VB6 / VBA / PowerBuilder / Delphi / 任何支持 ActiveX 容器的语言开发者** — 这是 Suite Pro 的核心受众，没有第二选择能这么全面

- **维护老 ActiveX 程序、想给它"换张脸"的工程师** — 给旧 COM 程序套上 Ribbon 和停靠窗格，性价比极高

- **做企业内部工具、需要 Office / VS 风格界面的中小团队** — 一套买断，2 台设备，团队摊下来不贵

- **做数据可视化看板的开发者** — Chart Pro 加 Report Control 加 Gauge，办公场景够用

- **不做 Visual C++ / MFC 的开发者注意** — MFC 开发者要的是同公司的 **Toolkit Pro**，别买错了

如果你是 .NET / WPF / WinUI 3 时代的纯新项目，老实说 Suite Pro 不是首选，那是另一套生态的事。

## 几个常被问到的点

**macOS 版有吗？** 没有，Suite Pro 只跑 Windows，没有 Mac 版。

**支持中文界面吗？** 最新版已正式支持中文界面，下载安装就能切。

**激活几台设备？** 单用户授权支持同时激活 2 台设备，参数以商品页说明为准。

**版本过期了还能用吗？** 能。授权是终身买断，过期只是不再享受新版本更新，已安装的版本永久可用。

**市面上有更便宜的"正版"靠谱吗？** 异常低价的注册码可能涉及盗刷、共享授权或政策滥用，开发商一旦识别会封禁，售后维权困难。建议走 ，来源可追溯到 Codejock 官方。
